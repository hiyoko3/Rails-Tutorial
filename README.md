## Usage


Downloading a project

```
$ git clone *** # Choose a repository
```

or

Downloading `Zip file` and extracting it.

Next, we need to install `RubyGems`.

```
$ bundle install --without production
```

Database migration.

```
$ rails db:migrate
```

A test to see this app works!

```
$ rails test
```

If you are ok?

Let's start an app!

```
$ rails server
```
